DaumView 플러그인은 다음뷰에서 제공하는 서비스를 워드프레스에서도 편리하게 사용할 수 있게 하기 위한 도구

== Description ==

본 플러그인은 <a href="http://v.daum.net/user/join">DaumView</a>에 가입하셔야 사용가능한 플러그인입니다.

* 포스트 작성시 다음뷰 전용 메타박스를 통해 원하는 채널에 포스트 송고

* 송고 후, 포스트의 상단 및 하단에 추천박스 출력

* 포스트, 페이지, 커스텀 포스트 타입의 편집 모드에서 모두 지원

* 다음뷰에서 제공하는 추천박스 3가지 타입 중 택 1 (http://daumview.tistory.com/32)

* 다음뷰의 총 4가지 위젯 제공
	
	1. MY글 위젯 : http://v.daum.net/my?exp=widget
	
	2. 구독 위젯 : http://widgetbank.daum.net/widget/view/739
	
	3. 랭킹 위젯 : http://widgetbank.daum.net/widget/view/434
	
	4. 추천 LIVE : http://widgetbank.daum.net/widget/view/395

== Installation ==

플러그인을 설치하시기 위해서 다음과 같은 순서로 진행하시면 됩니다.

1. DaumView 플러그인을 다운로드 받습니다.

2. 다운로드 받은 플러그인을 FTP를 통해 Plugins 폴더로 업로드 합니다.

3. 플러그인 관리에서 DaumView 플러그인을 활성화 시킵니다.

4. 설정 > DaumView로 접속합니다.

5. DaumView에 가입합니다. (가입하셨다면 다음 단계로 이동하세요)

6. DaumView 가입시 기입하신 블로그 주소를 작성합니다. (초기값으로 현재 사이트의 주소가 자동입력됩니다. 만약, 기본 블로그 주소와 틀린 경우 변경 입력하세요)

7. 블로그 주소가 일치하는 경우 하단에 옵션 메뉴가 나타납니다.

8. 옵션을 설정하고 사용합니다.

9. 위젯활성화시 제공하는 4개의 위젯을 사용하실 수 있습니다.

== Frequently Asked Questions ==

문의사항은 안반장의 개발 노트 블로그 http://qnibus.com/blog/daumview-plugin/ 에
댓글이나 페이스북 qnibus, 트위터 @qnibus로 의견 남겨주세요.

1. 송고하는 내 포스팅의 경로는 어떻게 되나요?
	* 워드프레스의 ShortURL을 전송합니다.

2. 송고한 다음뷰를 삭제할 수 있나요?
	* 원칙적으로 DaumView의 MyView로 접속하셔서 삭제해야하며, 삭제하시면 해당 포스팅의 송고 정보는 나타나지 않습니다.
	
3. Pretty URL을 사용중인데 플러그인 사용에는 문제 없나요?
	* 오히려 ShortURL을 사용해 퍼머링크가 변경되더라도 변경된 퍼머링크로 알아서 찾아가 줍니다.
